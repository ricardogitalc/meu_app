# Rotas da API

# LOGIN
ENVIAR MÁGIC LINK
- POST /auth/login
  - Body: { destination: string }
  - Retorno: { message: string, login_token: string, verify_url: string }

VERIFICAR MÁGIC LINK
- GET /auth/verify-login
  - Headers: { login-token: "login_token" }
  - Retorno: { message: string, user: user, jwt_token: string, refresh_token: string }

LOGIN GOOGLE OAUTH
- GET /auth/google
  - Inicia fluxo de autenticação Google

CALLBACK GOOGLE OAUTH
- GET /auth/google/callback
  - Callback do Google OAuth
  - Retorno: { message: string, user: user,  jwt_token: string, refresh_token: string }


### REGISTRO
# CADASTRAR USUÁRIO
- POST /users/register
  - Body: { 
      email: string,
      firstName: string, 
      lastName: string,
      whatsappNumber?: string 
    }
  - Retorno: { 
      message: string,
      register_token: string,
      verify_url: string 
    }

# VERIFICAR CADASTRO
- GET /auth/verify-register
  - Headers: { register-token: "register_token" }
  - Retorno: { message: string, jwt_token: string, verify_url: string }

# REFRESH TOKEN
- GET /auth/refresh-token
  - Headers: { refresh-token: "refresh_token" }
  - Retorno: { message: string, user: user, jwt_token: string }

## CRUD USUÁRIOS

### Listar Usuários (Admin)
- GET /users/all
  - Query: 
    - skip?: number
    - take?: number
  - Headers: Authorization: Bearer {jwt_token}
  - Retorno: Users[]

### Buscar Usuário
- GET /users/:id
  - Params: id: number
  - Headers: Authorization: Bearer {jwt_token}
  - Retorno: User

### Atualizar Usuário
- PATCH /users/:id
  - Params: id: number 
  - Body: {
      firstName?: string,
      lastName?: string,
      whatsappNumber?: string,
      imageUrl?: string
    }
  - Headers: Authorization: Bearer {jwt_token}
  - Retorno: { message: string, user: user, jwt_token: String }


### Deletar Usuário
- DELETE /users/:id
  - Params: id: number
  - Headers: Authorization: Bearer {jwt_token}
  - Retorno: { message: String, user: user }

### Schema prisma
model User {
  role           Role     @default(USER)
  id             Int      @id @default(autoincrement())
  email          String   @unique
  firstName      String
  lastName       String
  whatsappNumber String?
  imageUrl       String?
  verified       Boolean  @default(false)
  createdAt      DateTime @default(now())
  updatedAt      DateTime @updatedAt
}
